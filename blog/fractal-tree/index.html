Fractal trees in D3

Making your own fractal trees

Currently I’m in an amazing class entitled Chaotic Dynamics by a wonderfuly edifying instructor - Liz Bradly. Last week when we were studying fractals, I went a bit overboard and wrote a Javascript application that builds a Cantor-like set and then allows you to see the result of the set as a tree which as fractal properties.

Please play around with it here

If you have any ideas of how to extend this, please do get in touch. You can expect to see a lot more interesting visuals and projects in the coming months based around chaotic systems.
